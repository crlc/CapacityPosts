# Capacity Posts

## Instructions to Build
Build the project with
```
npm run build
```
Serve the page to access it via ```http://localhost:3000/```
```
npm install -g serve
serve -s build
```

## Instructions to Run
Run the app with
```
npm start
```
then open ```http://localhost:3000/``` to see your app.

## Instructions to Test
Test the appl with
```
npm test
```
